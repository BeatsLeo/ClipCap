## 基于ClipCap实现中文Image Caption

一个项目用于两个课程设计也挺不错`^_^`

### 图像特征提取

利用ClipCap在极大的数据集进行预训练所得到的图像特征提取模型

### 文本生成

利用GPT2在极大的数据集进行预训练所得到的文本生成模型

### 图像特征到文本特征映射网络

* MLP
* MLP + GPT2微调
* Transformer
* Transformer + GPT2微调

### 参考文献
